# Simon-Says
A fun and interactive web-based Simon Says. Test your memory by following an ever-growing sequence of colors.
This game improves players memory skills
Built with C++, HTML and JS
Learned HTML for this project using youtube and LLMs
The biggest bug was making it remember showed color after correct input
